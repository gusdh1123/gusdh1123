## 📚 About Me

- 🔍 단순 반복보다 **문제 해결 과정**에서 즐거움을 느끼며 개발자가 되었습니다.
- 💡 학원에서 컴퓨터 구조·메모리 처리 방식까지 배우며 개발에 대한 흥미가 더 깊어졌습니다.
- 👨‍💻 팀 프로젝트에서 게시글 CRUD, 소셜 로그인, Ajax 댓글, 좋아요/팔로우 등 다양한 기능을 직접 구현했습니다.
- ☁️ AWS EC2와 Docker 기반의 배포 환경 구성 경험으로 백엔드 전체 흐름을 이해했습니다.
- 🧠 콜센터·편의점 경험으로 다양한 상황에서 침착하게 대응하는 문제 해결력을 갖추었습니다.
- 🎭 연극 활동을 통해 발표력과 커뮤니케이션 능력을 키웠습니다.
- 📚 현재 컴퓨터공학 학위 과정과 자격증(정보처리산업기사)으로 실력을 강화하고 있습니다.
- 🌱 꾸준한 학습과 협업을 기반으로 팀과 함께 성장하는 개발자가 되고자 합니다.
