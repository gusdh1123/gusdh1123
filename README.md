## 🙋‍♂️ About Me

- 🔍 문제를 분석하고 해결하는 과정에서 즐거움을 느껴 개발자의 길을 선택했습니다.
- 💡 학원에서 컴퓨터 구조·메모리 처리 방식 등 기본 개념까지 학습하며 개발에 대한 이해와 흥미가 더욱 깊어졌습니다.
- 👨‍💻 팀 프로젝트에서 게시글 CRUD, 소셜 로그인(OAuth 2.0), Ajax 댓글·대댓글, 좋아요·팔로우·북마크 등의 기술을 구현했습니다.
- ☁️ AWS EC2(Ubuntu)에 Docker를 설치해 Oracle DB 컨테이너를 구성하고, 동일 서버에 Tomcat을 설치해 Spring Boot WAR 파일을 배포하는 과정도 진행하며 서비스가 동작하는 전체 흐름을 이해할 수 있었습니다.
- 🧠 콜센터·편의점 근무를 통해 다양한 상황에서도 침착하게 문제를 해결하는 능력을 갖추게 되었습니다.
- 🎭 연극 활동을 통해 발표력과 커뮤니케이션 능력을 키웠습니다.
- 🎓 현재 학점은행제 컴퓨터공학 학위 과정과 자격증 준비를 통해 기본기와 이론을 다지고 있습니다.
- 🌱 꾸준한 학습과 협업을 바탕으로 팀과 함께 성장하는 개발자를 목표로 하고 있습니다.
